## Passo a passo para rodar o projeto

1. Clone o projeto:

```
 git clone git@github.com:Thalis-Freitas/blog-app.git
```

2. Entre na pasta do projeto:

```
 cd blog-app/
```

3. Instale as dependências:

```
 yarn install
```

5. Suba o servidor:

```
 yarn serve
```

6. Acesse http://localhost:8080/
